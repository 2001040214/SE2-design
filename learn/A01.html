A01:2025 — Broken Access Control

OWASP Top 10 (2025) • Lesson page

What you’ll learn

  • Explain authorization vs authentication and where access control belongs in an app.
  • Identify common patterns: IDOR/BOLA, forced browsing, missing function-level checks.
  • Understand horizontal vs vertical privilege escalation with real request examples.
  • Apply deny-by-default and enforce checks on the server, not in the UI.
  • Test authorization using repeatable checklists and evidence-based test cases.
  • Implement defenses: ownership checks, RBAC/ABAC policies, centralized authorization, logging.
