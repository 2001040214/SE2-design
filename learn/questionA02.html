Question 1
Which example is a classic security misconfiguration?
  • A. Using parameterized SQL queries
  • B. Leaving debug mode enabled in production
  • C. Hashing passwords with a modern algorithm
  • D. Validating JWT signatures
Hint: Think of unsafe defaults.
Show answer
Correct: B. Leaving debug mode enabled in production
Explanation: Think of unsafe defaults.
Question 2
Why is permissive CORS with credentials risky?
  • A. It makes pages load slower
  • B. It can allow a malicious origin to read sensitive responses in a user’s browser
  • C. It prevents TLS from working
  • D. It disables logging
Hint: Browser enforces CORS rules.
Show answer
Correct: B. It can allow a malicious origin to read sensitive responses in a user’s browser
Explanation: Browser enforces CORS rules.
Question 3
