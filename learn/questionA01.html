Question 1
Which statement best describes authorization (not authentication)?
  • A. Verifying a user’s identity (who they are)
  • B. Deciding what an authenticated user is allowed to do
  • C. Encrypting data sent between client and server
  • D. Storing passwords securely with hashing
Hint: Think: permissions after login.
Show answer
Correct: B. Deciding what an authenticated user is allowed to do
Explanation: Think: permissions after login.
Question 2
An IDOR/BOLA issue most commonly happens when an app…
  • A. Uses HTTPS instead of HTTP
  • B. Validates input on the client only
  • C. Accepts an object ID and returns data without checking ownership/permissions
  • D. Requires MFA during login
Hint: Look for object IDs in URLs/JSON.
Show answer
Correct: C. Accepts an object ID and returns data without checking ownership/permissions
Explanation: Look for object IDs in URLs/JSON.
Question 3
Which is the safest default policy for access control?
  • A. Allow by default, deny only special cases
  • B. Deny by default, allow only explicitly permitted actions
  • C. Allow admins only, deny everyone else
  • D. Allow if the user is logged in
Hint: Default stance matters.
Show answer
Correct: B. Deny by default, allow only explicitly permitted actions
Explanation: Default stance matters.
Question 4
Horizontal privilege escalation refers to…
  • A. A user becoming an admin
  • B. A user accessing another user’s resources at the same role level
  • C. A server crash due to errors
  • D. Encrypting data at rest
Hint: Same role, different owner.
Show answer
Correct: B. A user accessing another user’s resources at the same role level
Explanation: Same role, different owner.
