A02:2025 — Security Misconfiguration

OWASP Top 10 (2025) • Lesson page

What you’ll learn

  • Define misconfiguration and why “secure defaults” matter.
  • Spot dangerous settings: debug mode, verbose errors, permissive CORS, open storage.
  • Understand configuration drift across environments (dev/stage/prod).
  • Build hardening baselines for servers, frameworks, containers, and cloud.
  • Test for misconfigurations using quick checks and automation-friendly rules.
  • Apply defense-in-depth: headers, least exposure, patching, and secure deployment.
